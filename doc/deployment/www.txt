* Web Server Dependencies

tomcat
java
jakarta-jstl
jdbc-postgresql / libpg-java
log4j


* Web Server Deployment Process

1. Install tomcat

2. Set listen address/port attributes

    File:       $CATALINA_HOME/conf/server.xml
    Element:    Connector
    Attributes: address, port

3. Disable AJP Connector
    
    File:       $CATALINA_HOME/conf/server.xml
    Attribute:  Connector

4. Configure database access logging
    
    File:       $CATALINA_HOME/conf/server.xml
    Element:    Valve
    Attribute:  className
    Value:      JDBCAccessLogValve

5. Configure application logging

    File:       $CATALINA_HOME/conf/logging.properties
    Property:   handlers
    Value:      JDBCAppender

6. Grant application permissions

    File:       $CATALINA_HOME/conf/policy.d/50user.policy
    Codebase:   file:/${catalina.base}/webapps/-
    Permission: SocketPermission (localhost:postgres, connect)
    Permission: SocketPermission (*, resolve)

6. Create a src/lib subdirectory under the working directory root

7. Create links to shared log4j, jdbc and jakarta taglib jar files from src/lib

8. Configure the application to point to an existing database

    File:       $WORKING_DIR/etc/autochef.properties

9. Build the application .war file, by executing 'ant'

10. Deploy the .war file to the root application location

    File: $CATALINA_HOME/webapps/ROOT.war

11. Start tomcat service
